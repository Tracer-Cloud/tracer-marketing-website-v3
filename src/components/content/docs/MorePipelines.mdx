# How to connect more pipelines?

Tracer combines pipelines from different instances into one platform. 
To connect a new instance to the platform, you can follow the quick steps (or watch the video) below.

_Note: Currently the Tracer platform does not distinguish between different pipelines within the same environment. It will display the metrics, flowsheets, and custom logs of the different runs, tools from the same environment (independent of the actual pipeline)._
