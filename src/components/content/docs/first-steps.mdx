<Steps>

### Sign up on [app.tracer.cloud](http://app.tracer.cloud) with Github or Google

### Copy the one-line integration into your terminal.

You can connect to the Tracer platform using one line copied into your terminal.

```bash
curl -sSL https://setup.tracer.cloud | bash -s -- <YOUR_API_KEY> && . ~/.bashrc && tracer help
```

_One line of code - don't forget to update the API key first!_

#### 2.1 How to find your API key?

Your first API key will be automatically generated when you follow the sign up steps.
